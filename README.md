Multi Pages App (MPA) for [Wakatobi Framework](https://github.com/ardhi/wakatobi)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install wakatobi-mpa
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```wakatobi-mpa``` in it
